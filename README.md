## 📌 About The Project

This project is a **next-generation personal portfolio** designed for an aspiring AI/ML Engineer. It combines **clean UI/UX, 3D effects, and dynamic animations** to create an engaging user experience.

The goal is to showcase:

* Technical skills
* AI/ML projects
* Education & certifications
* Contact & social presence

---

## ⚙️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Styling:** Custom CSS (no frameworks)
* **Fonts:** Google Fonts (Orbitron, Rajdhani, Share Tech Mono)
* **Icons:** Font Awesome
* **Animations:** CSS animations + JavaScript interactions
* **Canvas:** Particle background system

---

### 🎨 UI/UX
