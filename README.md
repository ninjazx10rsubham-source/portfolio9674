# 🚀 Subham Dey | AI/ML Engineering Portfolio

Welcome to my professional digital portfolio! This project is a highly animated, multi-page website designed to showcase my journey and technical expertise as a **Future AI/ML Engineer**.

---

## 👨‍💻 About Me
I am a B.Tech CSE (AI & ML) student at MIT Vishwaprayag University. My focus lies in Artificial Intelligence, Machine Learning, and leveraging modern AI tools to build efficient, smart systems.

---

## ✨ Features
* **3D Parallax Hero:** An interactive 3D profile card that reacts to mouse movements.
* **Terminal Typing Effect:** Dynamic typewriter animation for professional titles.
* **Verified Credentials:** A dedicated section for technical certifications with PDF verification links.
* **Glassmorphism UI:** Modern, translucent design with a high-tech "Cyberpunk" aesthetic.
* **Highly Responsive:** Optimized for a seamless experience across all device sizes.
